# Volunteer Platform

Платформа волонтерской помощи - React приложение для управления заданиями волонтеров.

## Технологии

- React 18
- React Router (HashRouter для GitHub Pages)
- Tailwind CSS
- Vite
- localStorage для хранения данных

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Роли и возможности

### ADMIN (Администратор)
- Просмотр всех заданий
- Одобрение/отклонение заявок (статус PENDING)
- Подтверждение выполнения заданий (статус REVIEW)

### USER (Пользователь)
Пользователь может выполнять обе функции: создавать задания и помогать другим.

**Вкладка "Лента помощи":**
- Просмотр доступных заданий (статус OPEN), созданных другими пользователями
- Взятие задания в работу
- Защита: нельзя взять свое собственное задание

**Вкладка "Мои задания":**
- Просмотр всех заданий, которые создал пользователь
- Создание новых заявок на помощь

**Вкладка "В работе":**
- Просмотр заданий, которые пользователь взял на выполнение
- Завершение задания (перевод в статус REVIEW)
- Отслеживание начисленных баллов

## Статусы заданий

1. **PENDING** - На проверке (создано пользователем)
2. **OPEN** - Поиск помощника (одобрено админом, видно всем пользователям)
3. **IN_PROGRESS** - В работе (взято пользователем)
4. **REVIEW** - На проверке выполнения (завершено пользователем)
5. **DONE** - Выполнено (подтверждено админом, начислены баллы)
6. **REJECTED** - Отклонено (админом)
